## Projects

### mini-file-audit

Repository: https://github.com/alexiskuypers/mini-file-audit

`mini-file-audit` is my first structured Python CLI project, built to validate Phase 1 of my backend roadmap.

It scans a folder, filters files, generates CSV and JSON reports, displays a terminal summary, and writes simple execution logs.

Main skills practiced:

* Python project structure with `src/`
* CLI development with `argparse`
* file handling with `pathlib`
* CSV and JSON report generation
* basic logging
* automated tests with `pytest`
* minimal packaging with `pyproject.toml`
* Git and GitHub workflow

Status: completed v1 as a Phase 1 learning project.

Known limits:

* no database
* no REST API
* no Docker
* no CI/CD
* simple error handling
* no advanced business logic
